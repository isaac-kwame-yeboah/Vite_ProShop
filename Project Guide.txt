         //  Project Guide  // 
             /*  Frontend Creation  */
1. npx create-react-app my-app   || npx create-react-app@latest frontend

2. $ ls -a  (List all git files in the frontend folder )
   $ rm -fr .git  (Remove the hidden .git repository file)
    Move the .gitignore file to the root folder 

3. Initialize a Git Repository in the root folder 
      git branch -M main  => Select branch 
      git push -u origin main   => Push to Github 

4. Install React Bootstrap, Bootstrap, React Icons
  => npm i react-bootstrap bootstrap@5.1.3 react-icons  

5. Bring in Bootstrap in index.js file 
   => import "bootstrap/dist/css/bootstrap.min.css"; 

6. Install React-Router-Dom
  => npm i react-router-dom

7. Install React Router Bootstrap 
    => npm i react-router-bootstrap 

8. Redux && Redux Toolkit :: 
     npm install @reduxjs/toolkit react-redux 

9. Install React Toastify;
      npm i react-toastify
 
10. npm i @paypal/react-paypal-js  => Install React Paypal
